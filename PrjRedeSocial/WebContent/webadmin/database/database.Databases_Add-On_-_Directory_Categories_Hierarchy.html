Directory application for creating a custom directory. For example to create a geographical directory for website visitors to select a continent, then a country, then a state and so on.
Select a directory name below. For example "Location" or "Directory". Then click the "Import Add-On Module" button below. The start page for your directory will be located in the content group with the directory name you select below. The name of your start page for your directory will also be the directory name you select below. For example if you type "Location" in the directory name field below the name of your directory start page will be "Location" and the page will be located in the "Location" page group. The entries you add to your directory will be located in a custom content database with the directory name you select below.
