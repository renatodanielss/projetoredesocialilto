Stores/Branches directory application for creating a searcheable list of stores/branches. When a store/branch is selected a map and details about the store/branch is displayed.
Select a Name for your directory of stores below. For example "Branches". Then click the "Import Add-On Module" button. The start page for your Stores/Branches directory will be located in the content group with the name you select below. The name of the start page for your Stores/Branches directory will also be the name you select below. For example if you type "Branches" in the name field the name of your Stores/Branches directory start page will be "Branches" and the page will be located in the "Branches" page group. The stores/branches you add to your Stores/Branches directory will be located in a custom content database with the name you select below.
