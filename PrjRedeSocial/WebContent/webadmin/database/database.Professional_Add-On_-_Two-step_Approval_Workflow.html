Two-step editor - manager - legal - publisher approval workflow for website editors to add and update content and keep it private until they request a manager and the legal department to approve the content and website publishers to publish the content.
The User Database, User Categories, Administrator Access Restrictions and Publishing features should also be enabled to use this workflow.
