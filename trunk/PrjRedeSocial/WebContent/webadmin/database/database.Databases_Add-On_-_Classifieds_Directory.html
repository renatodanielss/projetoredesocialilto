Classifieds application for creating a directory of classifieds. Enables visitors to submit classifieds and moderators to approve classifieds before they are published.
Select a Directory Name below. For example "Classified Directory" or "Ad Directory". Select a name for your Classifieds database in the Classifieds field below. For example "Classifieds" or "Ads". Then click the "Import Add-On Module" button. The start page for your Classified Directory will be located in the content group with the Classified Directory Name you select below. The name of the start page for your Classified Directory will also be the Classified Directory Name you select below. For example if you type "Classified Directory" in the Classified Directory Name field the name of your Classified Directory start page will be "Classified Directory" and the page will be located in the "Classified Directory" page group.
